# IT3040 – Assignment 1 | Playwright Automation Testing

## Project Overview
This project was developed as part of IT3040 – IT Project Management (Assignment 1).
It contains automated UI test cases implemented using Playwright to evaluate the
accuracy and usability of a Singlish-to-Sinhala transliteration web application.

## Tools & Technologies
- Playwright
- Node.js
- JavaScript

## Prerequisites
- Node.js (LTS version recommended)
- npm

## Installation Steps
1. Clone the repository:
   git clone https://github.com/MahenBandaraHiggoda/IT3040-singlish-to-sinhala-playwright-tests.git

2. Navigate to the project directory:
   cd IT3040-singlish-to-sinhala-playwright-tests

3. Install dependencies:
   npm install

4. Install Playwright browsers:
   npx playwright install

## Running the Tests
Run all tests using:
npx playwright test
