# IT3040 – Assignment 1 | Playwright Automation Testing

## Project Overview
This project was developed as part of IT3040 – IT Project Management (Assignment 1).
It contains automated UI test cases implemented using Playwright to evaluate
Singlish-to-Sinhala transliteration functionality.

## Tools & Technologies
- Playwright
- Node.js
- JavaScript

## Prerequisites
- Node.js (LTS recommended)
- npm

## Installation Steps
1. Clone the repository
2. Navigate to the project directory
3. Install dependencies using:
   npm install
4. Install Playwright browsers using:
   npx playwright install

## Running the Tests
Run all tests using:
npx playwright test
